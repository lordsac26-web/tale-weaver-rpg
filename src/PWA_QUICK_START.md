# PWA Quick Start Guide

## How to Install Tale Weaver App

### Method 1: Automatic Prompt (Easiest)
1. **Visit the app** and use it for 30+ seconds
2. **Wait for install banner** to appear (bottom-right corner)
3. **Click "Install"** button
4. **Confirm** in browser dialog
5. **Done!** App installed to desktop/home screen

### Method 2: Install Button
1. **Click "Install App"** button in the header
2. **Follow browser prompts** to install
3. **Done!**

### Method 3: Browser Menu (Manual)

**Chrome:**
- Click menu (⋮) → "Install Tale Weaver"
- Or: Menu → "More tools" → "Create shortcut" → Check "Open as window"

**Edge:**
- Click menu (⋯) → "Apps" → "Install Tale Weaver"

**Safari (Mac):**
- Click "File" → "Add to Dock..."

**Safari (iPhone):**
- Tap Share button → "Add to Home Screen"

**Chrome (Android):**
- Tap menu (⋮) → "Install" or "Add to Home screen"

## Check Installation Status

Look at the **PWA Debug panel** on the Home page (desktop only):
- Green ✓ = Working
- Yellow ⚠️ = Waiting/Loading
- Red ✗ = Issue detected

## Troubleshooting

### Don't see install option?
1. **Use the app for 30+ seconds** (browser needs engagement)
2. **Check the debugger** on Home page
3. **Try manual installation** via browser menu
4. **Use Chrome or Edge** for best support

### Install button doesn't work?
1. **Check debugger** - is install prompt ready?
2. **May already be installed** (check your apps)
3. **Browser may not support** programmatic install
4. **Use manual installation** instead

### Still having issues?
- See `PWA_TROUBLESHOOTING.md` for detailed help
- Check browser console (F12) for errors
- Try incognito/private mode
- Clear browser cache
